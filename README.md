# WaymakerLabs Claude Plugins

A collection of useful Claude Code plugins by WaymakerLabs.

## Installation

```bash
# Add marketplace
/plugin marketplace add waymakerlabs/claude-plugins

# List available plugins
/plugin list waymakerlabs-claude-plugins
```

---

## Plugins

### 1. minimal-statusline

Clean minimal statusline without progress bars - just gradient-colored percentages.

**Preview:**
```
Opus 4.5 | Explanatory | ~/Dev (main)✓
Context 4% | 5H 0% (3h43m) | 7D 14% (Fri)
```

**Features:**
- No progress bars, just clean percentage numbers
- Catppuccin theme gradient colors (color changes based on usage)
- 2-line layout: Model/Style/Path | Context/5H/7D usage

**Install:**
```bash
/plugin install minimal-statusline@waymakerlabs-claude-plugins
```
